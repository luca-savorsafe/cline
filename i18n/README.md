# Parrot 扩展多语言支持

## 概述

此目录包含 Parrot VS Code 扩展的本地化文件，支持英文和简体中文。

## 文件结构

```
i18n/
├── README.md (此文件)
├── bundle.l10n.json      # 语言包配置
├── package.i18n.json     # 英文语言包（默认）
└── zh-cn/
    └── package.i18n.json # 简体中文语言包
```

## 支持的语言

1. **英文 (en)** - 默认语言
2. **简体中文 (zh-cn)** - 中文翻译

## 本地化字符串

已本地化的字符串包括：

### 扩展元数据
- `extension.displayName` - 扩展显示名称
- `extension.description` - 扩展描述

### 引导教程 (Walkthrough)
- 教程标题和描述
- 5个步骤的标题和描述

### 命令 (Commands)
- 所有用户界面命令的标题
- 包括：新建任务、MCP服务器、历史记录、账户、设置等

### 配置
- 配置标题

### 视图
- 视图标题

## 使用方法

### 在 VS Code 中测试
1. 将 VS Code 语言设置为简体中文
2. 安装或重新加载扩展
3. 扩展界面将显示中文翻译

### 添加新语言
1. 在 `bundle.l10n.json` 中添加新语言代码
2. 创建对应的语言目录（如 `zh-tw/`）
3. 复制 `package.i18n.json` 到新目录并翻译所有字符串

### 添加新的本地化字符串
1. 在 `package.i18n.json` 中添加新的键值对
2. 在所有语言包文件中添加对应的翻译
3. 在代码中使用本地化键引用字符串

## 技术细节

### package.json 配置
```json
{
  "localization": "./i18n"
}
```

### 语言包格式
语言包使用 JSON 格式，键名采用点号分隔的命名约定：
- `category.subcategory.name` 格式
- 保持一致的命名规范

## 维护指南

1. **更新翻译**：直接编辑对应的语言包文件
2. **添加新字符串**：确保所有语言包都同步更新
3. **测试**：在更改后测试扩展的本地化显示
4. **质量检查**：确保翻译准确、一致，符合技术术语规范
